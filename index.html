Top 100

Choose a file and get a list of the top 100 most frequently occurring words...

How to use...

Follow these steps and find out the top 100 most frequently occurring words from a text file:

  • Click to "Choose File" near the top.
  • Select a file.
  • Explore a list of words from most occurances to least.

Things to note...

  • The list returns only the top 100 frequently occurring unique words.
  • It's ok if the file does not contain more than 100 unique words. An appropriate list will be created.
  • If you want to get a list from another text file simply click on "Choose File" again and select the desired file.
  • Stop-Words will be NOT be on the list. (For more information see below)
